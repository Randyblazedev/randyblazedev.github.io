# RBlaze Portfolio

Personal portfolio website built from scratch on mobile — showcasing projects, skills, tools, and a bit of personality.

## Built With

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Typed.js, AOS)
- Devicons
- Google Fonts (JetBrains Mono)

## Features

- Dark theme with blue glow effects
- Typing animation on hero heading
- Scroll reveal animations (AOS)
- Project cards with hover glow
- Toggle-able coding story
- Gaming section (Hands off the keyboard)
- Contact form (FormSubmit)
- OG meta tags for social sharing
- Google Search Console verified

## What I Learned

- Responsive layout with CSS Grid & Flexbox
- Integrating third-party libraries (Typed.js, AOS)
- Custom CSS animations (keyframes)
- CLI vs CDN for Tailwind CSS
- OG tags for social preview
- Deploying to Vercel
